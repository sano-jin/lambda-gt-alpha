# lambda-gt-alpha

A reference interpreter of the lambda GT language.

## Getting Started

### Prerequisites

- [opam](https://opam.ocaml.org/)

### Installation

```bash
git clone git@gitlab.ueda.info.waseda.ac.jp:sano/lambda-gt-alpha.git
cd lambda-gt-alpha/lambda_gt_alpha
opam install .
dune build
```

## Usage

```bash
./run example/dlist.lgt
```
